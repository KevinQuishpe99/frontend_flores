# Variables de Entorno para Vercel

Configura estas variables en el Dashboard de Vercel → Settings → Environment Variables
O durante el deploy con: `vercel env add`

```env
# Backend API URL (URL de Render o Railway)
VITE_API_URL=https://TU_BACKEND.onrender.com/api
# O si usas Railway:
# VITE_API_URL=https://TU_BACKEND.up.railway.app/api

# Google OAuth Client ID
VITE_GOOGLE_CLIENT_ID=TU_GOOGLE_CLIENT_ID_AQUI
```
